# THE COMPLETE STRENGTH PROTOCOL

**Full-Body Strength • Time-Efficient • Evidence-Based**

---

## 1. PROTOCOL OVERVIEW

### Design Principles
This protocol uses standing, bench-supported, and equipment-based movements exclusively. All exercises can be performed without floor contact, allowing consistent execution in any gym environment.

### Key Parameters
| Parameter | Value | Rationale |
|-----------|-------|-----------|
| **Frequency** | 3 sessions/week | Optimal recovery for natural lifters |
| **Duration** | 55–60 minutes | Sufficient volume without diminishing returns |
| **Rest Between Sessions** | 48–72 hours | Muscle protein synthesis window |
| **Movements** | 7 core + 2 optional | Full coverage with flexibility |

### Time Breakdown

| Phase | Duration | Purpose |
|-------|----------|---------|
| Warmup | 5 min | Mobility, activation, heart rate elevation |
| Strength | 42–50 min | Progressive overload on compound movements |
| Conditioning | 5 min | Cardiovascular maintenance |
| Recovery | 3 min | Decompression and parasympathetic activation |
| **Total** | **55–63 min** | Depends on exercise selection |

### Muscle Coverage Matrix

| Movement Pattern | Primary Muscles | Secondary Muscles |
|------------------|-----------------|-------------------|
| Hinge | Hamstrings, Glutes, Erectors | Grip, Core |
| Knee | Quadriceps, Glutes | Core, Adductors |
| Horizontal Push | Pectorals, Anterior Deltoids | Triceps |
| Vertical Push | Deltoids (all heads) | Triceps, Upper Traps |
| Horizontal Pull | Lats, Rhomboids, Rear Deltoids | Biceps, Grip |
| Vertical Pull | Lats, Teres Major | Biceps, Grip |
| Carry | Traps, Grip, Core | Full body stabilization |
| Calves | Gastrocnemius, Soleus | Ankle stabilizers |

## 2. PROGRESSION SYSTEM

### A. Recovery Assessment

Assess before each session to determine training intensity.

| State | Criteria | Action |
|-------|----------|--------|
| **Green** | 7+ hours sleep, no acute pain or illness | Train at programmed weights |
| **Yellow** | 5–6 hours sleep, minor fatigue or stiffness | Reduce all weights by 10% |
| **Red** | <5 hours sleep, acute pain, or illness | Skip lifting; light walk only |

### B. Weight Progression

| Outcome | Definition | Next Session |
|---------|------------|--------------|
| **Success** | Completed all sets and reps with controlled form | Add 5 lbs |
| **Partial** | Missed 1–2 reps or minor form breakdown | Repeat same weight |
| **Failure** | Missed 3+ reps or significant form breakdown | Repeat same weight |
| **Stall** | Same weight failed 3 consecutive sessions | Reduce weight by 10% |

### C. Scheduled Deload

Every 6th week, reduce all working weights to 60% regardless of performance. This allows connective tissue recovery and prevents accumulated fatigue.

### D. Missed Sessions

| Scenario | Protocol |
|----------|----------|
| Missed 1 session | Resume next available day (maintain 48hr minimum gap) |
| Missed 2+ sessions | Resume schedule; reduce weights 10% for first session |
| Illness with fever | Full rest until 24 hours fever-free; reduce weights 20% |
| Localized injury | Skip aggravating movement only; perform remaining exercises |

## 4. PHASE 2: STRENGTH TRAINING

### Execution Parameters
- **Tempo:** 3 seconds eccentric (lowering), 1 second concentric (lifting)
- **Rest:** 90 seconds between sets
- **Breathing:** Exhale on exertion, inhale on return

### Time Per Exercise
Each exercise takes approximately 5 minutes:
- Working time: 3 sets × 35–60s = 2 min
- Rest time: 2 intervals × 90s = 3 min

## 7. DIGITAL TRACKING

This protocol integrates with **Flexx Files**, an offline-capable progressive web app.

### Features
- Offline operation (no internet required)
- Recovery state enforcement at session start
- Automatic +5 lb progression on successful lifts
- Stall detection after 3 consecutive failures
- Deload prompts every 6th week
- Plate calculator for barbell exercises
- Form video access during workouts
- JSON export/import for data backup

### Installation
1. Open the app URL in a mobile browser
2. Select "Add to Home Screen" from browser menu
3. Launch from home screen for standalone experience
